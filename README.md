## ğŸ§ª Tentang Katalon Studio

**Katalon Studio** adalah platform automation testing yang mudah digunakan dan mendukung berbagai jenis pengujian, seperti:

- Web Testing  
- API Testing  
- Mobile Testing  
- Desktop Testing  

**Kelebihan Katalon Studio:**

- Mendukung *record & playback*  
- Scriptless automation, namun tetap bisa menggunakan Groovy/Java  
- Dukungan Test Case, Test Suite, Test Suite Collection  
- Integrasi Git, CI/CD, Jenkins, Jira  
- Mudah digunakan untuk pemula maupun profesional

## ğŸ¯ Tujuan Pembuatan Automation

Automation testing pada SUMU Web dibuat untuk:

- Mempercepat proses pengujian rutin pada fitur utama  
- Mengurangi risiko kesalahan manusia (human error)  
- Menjamin stabilitas fitur melalui pengujian otomatis  
- Mempermudah dokumentasi hasil testing  
- Menyiapkan integrasi CI/CD di masa mendatang
